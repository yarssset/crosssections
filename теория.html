Метод следов в построении плоских сечений многогранников

Сущность метода следов при решении задач на построение сечений многогранников заключается в построении общих точек, а по ним и прямых пересечения (следов) секущей плоскости с плоскостями граней, диагональных или осевых сечений. Обычно решение задачи на построение сечения методом следов начинается с построения прямой пересечения секущей плоскости с основной плоскостью. Эту прямую называют основным следом.

Алгоритм построения сечения многогранника по методу следов:

1.Выбрать метод проецирования (центральное – пирамида; параллельное – призма)

2. Выяснить имеются ли в одной грани две точки сечения (если да, то через них можно провести сторону сечения)

3. Построить основной след секущей плоскости:

• Построить проекции данных точек на основную плоскость;

• Построить три прямые, проходящие через три данные точки;

• Построить три прямые, проходящие через проекции данных точек;

• Построить точки пересечения каждой прямой и соответствующей ей проекции;

• Все три точки лежат на одной прямой. Эта прямая является основным следом секущей плоскости.

(Примечание: так как прямая задается двумя точками, то достаточно провести любые две из трех пар прямых)

4. Построить след секущей плоскости на одной из граней многогранника

• Найти точку сечения на грани многогранника.

• Продолжить сторону основания той грани, в которой есть точка сечения, до пересечения со следом.

• Через полученную дополнительную точку на следе и точку сечения провести прямую, отметить точки пересечения её с рёбрами грани.

• Соединить точки отрезком. Получим сторону сечения

5.При необходимости повторить п.4.

Метод внутреннего проектирования в построении плоских сечений многогранников

Сущность метода внутренних проекций при решении задач на построение сечений заключается в том, что по проекциям точек секущей плоскости на основную плоскость находятся дополнительные точки секущей плоскости. При этом проекции искомых точек секущей плоскости выбираются так, чтобы они были связаны с проекциями точек, определяющих секущую плоскость, и чтобы число графических операций при решении задач было минимальным.

Этот метод является в достаточной мере универсальным. В тех случаях, когда нужный след (или следы) секущей плоскости оказывается за пределами чертежа, этот метод имеет даже определенные преимущества.

Метод переноса секущей плоскости в построении плоских сечений многогранников

Суть этого метода состоит в следующем: строится такое вспомогательное сечение данного многогранника, которое удовлетворяет следующим требованиям:

1) Оно должно быть параллельно секущей плоскости.

2) В пересечении с поверхностью данного многогранника образуется треугольник. После этого искомое сечение строится на основании свойств прямых, по которым две параллельные плоскости пересекаются третьей плоскостью.
